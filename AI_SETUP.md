# 🤖 Настройка AI-Помощника

## 1. Получить OpenAI API Key

1. Зарегистрируйтесь на https://platform.openai.com
2. Перейдите в раздел API Keys
3. Создайте новый ключ
4. Скопируйте его (начинается с `sk-...`)

## 2. Добавить в Netlify

1. Откройте: https://app.netlify.com/sites/hqf/configuration/env
2. Нажмите "Add a variable"
3. Имя: `OPENAI_API_KEY`
4. Значение: ваш ключ `sk-...`
5. Нажмите "Save"
6. Нажмите "Trigger deploy" для перезапуска

## 3. Пополнить баланс OpenAI

1. Откройте: https://platform.openai.com/settings/organization/billing
2. Добавьте карту
3. Пополните на $5+ (хватит на ~100-200 операций)

## 4. Готово!

Теперь во вкладке "🤖 AI-Помощник" можно:
- Загружать фото кроссовок
- Использовать голосовые команды
- Писать текстовые запросы

AI автоматически распознает модель и заполнит форму!

## Стоимость

- Vision (фото): ~$0.01 за запрос
- GPT-4 (текст): ~$0.001 за запрос
- ~$0.01-0.02 за операцию

## Модель AI

Используется **gpt-4o** - самая мощная мультимодальная модель OpenAI (2024-2026).

### Стоимость:
- Input: ~$2.50 за 1M токенов
- Output: ~$10 за 1M токенов
- Средний запрос с фото: ~1000-2000 токенов = $0.002-0.004
- Примерная стоимость: **$0.10-0.50 в день** при активном использовании

## Курсы валют

Курсы валют настраиваются вручную в файле `netlify/functions/ai-assistant.ts`:

```typescript
const EXCHANGE_RATES = {
  USD_TO_BYN: 3.20,  // Обновите с http://myfin.by/
  EUR_TO_BYN: 3.50,
  RUB_TO_BYN: 0.035,
  CNY_TO_BYN: 0.44,
};
```

Рекомендуется обновлять курсы раз в неделю.
